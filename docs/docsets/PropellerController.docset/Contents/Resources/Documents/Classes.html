  • GeneralCollectionController

    Handles common UICollectionViewDelegate and UICollectionViewDataSource methods for a UICollectionView

    controller.collectionView = collectionView
    controller.setDataSource(["Item One", "Item Two"])
    controller.willDisplayCell = { cell, data, _ in
        cell.textLabel.text = data

  • GeneralTableController

    Generic TableView Controller requiring Cell and Data associated types to be specified. Define class like so: GeneralTableController<SomeCellType, SomeDataType>. Using as below:

    static var nameSelection: (UITableView) -> GeneralTableController<NameCell, NameData> = { table in
   let controller = GeneralTableController<NameCell, NameData>()
   controller.tableView = table

   controller.willDisplayCell = { cell, data, iPath in
       cell.name = data.first
       cell.image.asyncSetImage(data.imagePath)
       cell.location = "row:" + iPath.row
   }
   return controller
}

  • GeneralMultiSelectionTableController

    Subclass of GeneralTableController that allows you you keep track of selected dataSource entry. Requires DataType to be Hashable. Also, you can use toggleSelection function to manually invert a data value’s selection state.
